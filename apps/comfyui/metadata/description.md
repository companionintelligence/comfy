# ComfyUI

A powerful and modular stable diffusion GUI and backend. ComfyUI provides a node-based interface for creating complex workflows.

## Features

- Node-based workflow interface
- GPU acceleration support
- Modular architecture
- Custom nodes support
- API access
- WebSocket support

## Configuration

- **Ollama Models Path**: Optional path to Ollama models
- **Extra Model Paths**: Additional paths for models
- **Enable GPU**: Enable GPU acceleration (default: true)
